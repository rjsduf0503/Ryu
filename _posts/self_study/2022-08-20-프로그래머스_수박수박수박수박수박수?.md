# 프로그래머스 LEVEL 1(수박수박수박수박수박수?)

  ![image](https://user-images.githubusercontent.com/34560965/185521168-454600b8-d1f9-4e3b-a942-c017b3e59d65.png)

  - 사용 언어 : javascript

  - 해결 날짜 : 2022-08-19

  - 해결 방법 :

    -  입력받은 길이만큼 for문 반복하여 홀수 인덱스일 때 '수', 짝수 인덱스일 때 '박' 추가

  - 코드

    ```javascript
    function solution(n) {
        var answer = '';
        for (var i = 1; i <= n; i++) {
            if (i % 2 === 1) answer += '수';
            else answer += '박';
        }
        return answer;
    }
    ```
